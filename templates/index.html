Welcome to the Callback Collector!

Introduction

This web site is used to demonstrate creating a request for, and capturing of, OAuth token and OAuth Verifier strings that come from Twitter once an account has granted permission to an application.

Click the button to authorize this app to collect data on behalf of your account

Unless modified, the url for the callback should be in this form: http://{PublicIP}:{{ port }}/callback?oauth_token=KEYVALUE&oauth_verifier=KEYVALUE2
